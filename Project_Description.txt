Description:
In this project of single player game, the player needs to pass a ball from starting to end point avoiding obstacles or barriers in dot matrix using push button. Clash of ball with obstacles will result in game over. However, winning a game the player can go to next level. Initially two levels are designed in the game. 

Key Features:
* Single player game
* 8x8 grid game (8x8 dot matrix)
* 1 push button to pass over the ball(point)
* Several levels

Components:
* ATmega16 microcontroller
* 8x8 dot matrix
* 5V fixed 1A positive power supply adapter
* Resistors
* Decoders
* Push Button
